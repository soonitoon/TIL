TIL-BOOK 페이지 🚀

로컬 실행

변경 사항이 생길 때마다 서버가 자동으로 재시작됩니다.

$ git clone https://github.com/soonitoon/TIL
$ cd ./TIL
$ npm i
$ npm start

빌드 및 배포

gitbook 빌드, 커밋, 저장소 push, 배포가 자동으로 이루어집니다.

$ npm run publish
